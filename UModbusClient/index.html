Method Descriptions

UModbusClient* Create(UModbusTcpSocket* InPort, int32 TimeoutMs) static link

Constructor. Creates a UModbusTcpSocket if the input pointer InPort is null.

bool Init(UModbusTcpSocket* InPort, int32 TimeoutMs) link

Attaches InPort and initializes the socket. Called by the constructor.

bool Connect(const FString& Host, int32 Port) link

Calls UModbusTcpSocket.Connect() on its attached socket.

void Disconnect() link

Calls UModbusTcpSocket.Close() on its attached socket if it was attached.

ModbusError ReadHoldingRegister(int32 Address, int32& OutValue) link

Writes the contents of the Modbus register at address to OutValue.

ModbusError ReadManyHoldingRegisters(int32 Address, int32 Count, TArray<int32>& OutValue) link

Writes the contents of the registers starting from address and ending at address + count to OutValue.

ModbusError WriteHoldingRegister(int32 Address, int32 Value) link

Writes Value to the register at the address.

ModbusError WriteManyHoldingRegisters(int32 Address, TArray<int32> Values) link

Writes Values to the registers between address and address + Values.size().

ModbusError ReadCoil(int32 Address, bool& OutValue) link

Writes the contents of the Modbus coil at address to OutValue.

ModbusError ReadManyCoils(int32 Address, int32 Count, TArray<bool>& OutValue) link

Writes the contents of the coils starting from address and ending at address + count to OutValue.

ModbusError WriteCoil(int32 Address, bool Value) link

Writes Value to the coil at the address.

ModbusError WriteManyCoils(int32 Address, TArray<bool> Values) link

Writes Values to the coils between address and address + Values.Num().

ModbusError ReadInputRegister(int32 Address, int32& OutValue) link

Writes the contents of the Modbus input register at address to OutValue.

ModbusError ReadManyInputRegisters(int32 Address, int32 Count, TArray<int32>& OutValue) link

Writes the contents of the input registers starting from address and ending at address + count to OutValue.

ModbusError ReadDiscreteInput(int32 Address, bool& OutValue) link

Writes the contents of the Modbus discrete input (or "contact") register at address to OutValue.

ModbusError ReadManyDiscreteInputs(int32 Address, int32 Count, TArray<bool>& OutValue) link

Writes the contents of the Modbus discrete input (or "contact") registers starting from address and ending at address + count to OutValue.

void Client_DebugPrint() link

Debug Function that outputs to LogTemp whether the socket is valid (NULL) or not.

void BeginDestroy() override virtual link

Ensures that the socket is disconnected before destroying this object. Overrides UObject.BeginDestroy()
