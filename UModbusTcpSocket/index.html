UModbusTcpSocket* CreateSocket() static link

Creates a UModbusTcpSocket.

void BeginDestroy() override virtual link

Ensures that the socket is closed before the object is freed.

bool Connect(const FString& Host, int32 Port) link

Uses ISocketSubsystem to create a socket and configure it. localhost is replaced with 127.0.0.1. Returns true if the port sucessfully connects.

int32 Read(uint8* Data, int32 Length, int32 TimeoutMs) link

Read data from the socket. Returns the number of bytes read.

int32 Write(const uint8* Data, int32 Length) link

Writes Length bytes from Data onto the socket. Returns the number of bytes written if sucessful and -1 if it fails.

void Close() link

Closes the socket.
